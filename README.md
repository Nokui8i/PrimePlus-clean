## Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL
- FFmpeg

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables
4. Initialize the database:
   ```bash
   npm run db:setup
   ```
5. Start the development server:
   ```bash
   npm run dev
   ```
